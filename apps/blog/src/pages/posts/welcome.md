# Welcome to Minions Bundle: Crm

We're excited to introduce **Minions Bundle: Crm** — CRM pipeline bundle — leads, deals, interactions, sequences.

## What's Included

- **TypeScript SDK**: `npm install @minions-bundles-crm/sdk`
- **Python SDK**: `pip install minions-bundles-crm`
- **CLI**: `npm install -g @minions-bundles-crm/cli`
